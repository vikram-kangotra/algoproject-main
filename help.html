Tips for using PRmePRed

  1. Protein sequence input length should not be less than 50 amino acids.
  2. When searching for proteins in raw or FASTA format, use the single-letter amino acid code.
  3. Please ensure that the sequences are in FASTA format before performing prediction on a file with multiple sequences.
